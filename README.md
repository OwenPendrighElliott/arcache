# cache.rs

A crate which implements a variety of caches with different eviction policies. All cache implementations are thread-safe and can be used in a multi-threaded environment.

## Implemented caches

+ `LRUCache`
+ `MRUCache`
+ `TTLCache`
+ `FIFOCache`

### Coming soon

+ `LFUCache`
+ `ARC`
